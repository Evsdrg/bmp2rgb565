#  bmp2rgb565
一款基于Python、由AI生成的，用于将任意bmp转为RGB565或者其他格式，并以数组形式输出，以供单片机彩色显示屏使用的程序。

------------
## 主要功能
将任意bmp图片转为RGB565，且可选择输出为16位数组或者8位数组，以适应不同的屏幕驱动。
还可以转为RGB332或者GRAY8灰度，并以8位数组模式输出。
输出的数组可以选择**小端序**或者**大端序**。
